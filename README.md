Instructions:

Run the simulation by running:
$ python3 Graph.py

you can change 3 variables through argparse inputs:
1. number of agents: -a
2. connectivity per agent: -c
3. degree of shared knowledge: -n

Example
$ python3 Graph.py -a 50 -c 10 -n 10
